Zero of Functions (ZOF) Solver

This file exists to satisfy the required project structure. The interactive Flask template lives inside /templates/index.html.

Getting Started

  1. Install dependencies with pip install -r requirements.txt.
  2. Run the CLI via python ZOF_CLI.py.
  3. Launch the web app with flask --app app run and visit http://127.0.0.1:5000.

The hosted link should be recorded in ZOF_hosted_webGUI_link.txt once deployment is complete.

For grading, please use the Flask interface rendered from templates/index.html.
